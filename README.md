# Midterm project - TFY4240 Electromagnetic theory
Numerical physics project, numerically solving rectangular 2d Laplace equation for arbitrary upper potential, in Python. Laplace equation solved by seperation of variables, and solution for general potential is written as a Fourier series. Fourier coefficients are determined by numerical integration. Use of NumPy, SciPy and MatPlotLib libraries. Quiver plots of derived electric field in report could be scaled down and adjusted in space for better visualization, the revised plot are given here as .png files.
